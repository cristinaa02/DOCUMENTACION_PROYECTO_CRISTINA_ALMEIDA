## 📁 Contenido del Repositorio

### 1. Documentación Principal
* **[PA Documentacion (Cristina Almeida).pdf](./PA%20Documentacion%20(Cristina%20Almeida).pdf)**: Archivo central con los enlaces a toda la memoria técnica y los manuales.

### 2. Entorno Linux e Infraestructura
* **Carpeta [PA_PROYECTO_V2/](./PA_PROYECTO_V2/)**: Contiene la infraestructura principal, los archivos de configuración.

    * **Código Python**: Archivos como `conector.py` para la comunicación con los datos.
    * **Vagrant**: Archivo `Vagrantfile` para el despliegue del entorno.

### 3. Entorno Windows e Interfaz
* **Carpeta [PA_WIN/](./PA_WIN/)**: Archivos destinados a la parte en Windows.
